About the AI Model

Text classification: BERT-based classifier, cross-checked against live web evidence (GDELT, Wikipedia, NewsAPI) with an LLM judge before returning a verdict. Fact-check evidence: GDELT + NewsAPI + Wikipedia + Google Fact Check.
